# 瑞普数据同步系统 实施计划（V2）

## 1. 里程碑

### M1：文档与数据结构升级
1. 输出 V2 需求文档、项目方案、实施计划。
2. 新增数据库迁移（source_type、system_settings、mssql_form_registry）。
3. 验收：服务可启动，迁移成功。

### M2：MSSQL 表单管理后台
1. 菜单改造：`表单管理` -> `氚云表单管理`。
2. 新增 `MSSQL表单管理` 页面。
3. 新增“表单库”配置弹窗与保存逻辑。
4. 验收：可配置路径并展示 MSSQL 表单列表。

### M3：MSSQL SQL 文件发现与导入
1. 递归扫描配置路径 `.sql` 文件。
2. 解析文件名并建立“表单-文件队列”。
3. 首次导入自动建档，后续增量导入。
4. 手动同步能力打通。
5. 验收：基于样例备份可完成初始化+增量同步。

### M4：联调与稳定性
1. 同步日志、错误日志完善。
2. 数据查看/筛选/分页验证。
3. 基础回归：氚云原功能不回退。
4. 验收：关键流程通过，日志可追溯。

### M5：企业资料库一期能力
1. 新增 `企业资料库` 页面与上传建表入口。
2. 完成 `Excel/CSV/TXT/DOCX/PDF` 文件解析与建表。
3. Excel 按 `Sheet` 拆分建表，一个 `Sheet` 一个表单。
4. 验收：可从样例资料文件生成企业资料表并出现在后台列表。

### M6：企业资料库编辑与导入增强
1. 资料表字段设置：支持修改字段显示名与备注。
2. 资料表内容页支持新增、编辑、删除内容。
3. 新增表内结构化导入能力，并增加字段格式校验。
4. 验收：可向现有资料表追加导入符合字段格式的数据，校验失败时拒绝导入。

## 2. 开发顺序（本轮）
1. 先做迁移与模型。
2. 再做仓储与服务。
3. 再做后台页面和路由。
4. 最后做编译与冒烟验证。
5. 企业资料库按“建表导入”与“表内追加导入”两条链路分别验证。

## 3. 测试清单
1. 路径为空、路径不存在、路径无权限。
2. 非法命名 SQL 文件处理。
3. FULL + INCR 混合导入顺序。
4. 重复同步幂等。
5. 手动同步与自动同步日志差异。
6. 中文备注展示与 API 透传。
7. Excel 多 Sheet 是否按 Sheet 正确拆分为多个表单。
8. 资料表字段显示名编辑后，页面与导出是否一致。
9. 资料表追加导入时，字段数量、顺序、名称不一致是否被正确拦截。

## 4. 上线准备
1. Ubuntu 环境变量模板补全。
2. systemd 服务脚本更新。
3. 数据库备份策略确认。
4. 管理员操作手册更新。
5. 企业资料库文件存储目录与清理策略确认。
