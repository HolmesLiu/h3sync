# 瑞普数据同步系统 实施计划（V2）

## 1. 里程碑

### M1：文档与数据结构升级
1. 输出 V2 需求文档、项目方案、实施计划。
2. 新增数据库迁移（source_type、system_settings、mssql_form_registry）。
3. 验收：服务可启动，迁移成功。

### M2：MSSQL 表单管理后台
1. 菜单改造：`表单管理` -> `氚云表单管理`。
2. 新增 `MSSQL表单管理` 页面。
3. 新增“表单库”配置弹窗与保存逻辑。
4. 验收：可配置路径并展示 MSSQL 表单列表。

### M3：MSSQL SQL 文件发现与导入
1. 递归扫描配置路径 `.sql` 文件。
2. 解析文件名并建立“表单-文件队列”。
3. 首次导入自动建档，后续增量导入。
4. 手动同步能力打通。
5. 验收：基于样例备份可完成初始化+增量同步。

### M4：联调与稳定性
1. 同步日志、错误日志完善。
2. 数据查看/筛选/分页验证。
3. 基础回归：氚云原功能不回退。
4. 验收：关键流程通过，日志可追溯。

## 2. 开发顺序（本轮）
1. 先做迁移与模型。
2. 再做仓储与服务。
3. 再做后台页面和路由。
4. 最后做编译与冒烟验证。

## 3. 测试清单
1. 路径为空、路径不存在、路径无权限。
2. 非法命名 SQL 文件处理。
3. FULL + INCR 混合导入顺序。
4. 重复同步幂等。
5. 手动同步与自动同步日志差异。
6. 中文备注展示与 API 透传。

## 4. 上线准备
1. Ubuntu 环境变量模板补全。
2. systemd 服务脚本更新。
3. 数据库备份策略确认。
4. 管理员操作手册更新。
